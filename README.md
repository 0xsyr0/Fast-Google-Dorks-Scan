# Fast Google Dorks Scan

The simple program for finding the common admin panels, the widespread file types and path traversal. The 100% automated.

Usage example:
--------------
```
chmod +x FGDS.sh
./FGDS.sh megacorp.one
```
or
```
bash ./FGDS.sh megacorp.one
```
